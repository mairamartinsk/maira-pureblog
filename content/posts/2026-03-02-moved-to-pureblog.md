Migration completed! This little blog is now running on [PureBlog](https://pureblog.org/), by [Kev Quirk](https://kevquirk.com/). He did an incredible job with this tiny cms.

First I migrated my mom's blog from HTMLy to PureBlog. It took me longer to work on a basic translation config for her, than to migrate the data. The migration went really well. [Her site](https://mariza.blog) is up and running with all the photos and tags and everything (and fully in Brazilian Portuguese, even the admin panel!). In the middle of the migration, Kev launched [Pure Comments](https://comments.purecommons.org/). I added that too and translated everything to pt-BR.

Then it was my turn! I was more or less happy with micro.blog. There were issues, but mostly happy. But now I had played for a few days with PureBlog and it is so cool, so simple, so minimal, just the right amount of everything and zero fluff. It is, honestly, a joy to use!

Moving away from micro.blog was much easier than I anticipated. All I had to do was write a couple scripts to move files around, reformat frontmatter, rename images. The last couple days I put some finishing touches, added a [/tags](/tags) page and an [/archive](/archive) page using custom routes. Next up: create a list of books read using data files.
